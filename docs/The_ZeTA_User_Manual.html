3.2 The document modelðŸ”—

  • Each document is comprised of multiple subdocuments. Some subdocument continue onto multiple pages (e.g. common header/footer, main content) and some dont.

  • A model for a subdocument contains the following things:

    • A width.

    • Whether this subdocument is open-bottom or not. "Open-bottom" in ZeTA jargon means it does not have a fixed height.

    • Margins of the four sides.

    • The font map used within this subdocument.
